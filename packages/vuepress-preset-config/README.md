阮喵喵的 vuepress 项目预设，为了快速初始化 vuepress 文档项目，故提供一系列 vuepress 的项目预设。

## 安装

<!-- automd:pm-install name="@ruan-cat/vuepress-preset-config" dev -->

```sh
# ✨ Auto-detect
npx nypm install -D @ruan-cat/vuepress-preset-config

# npm
npm install -D @ruan-cat/vuepress-preset-config

# yarn
yarn add -D @ruan-cat/vuepress-preset-config

# pnpm
pnpm add -D @ruan-cat/vuepress-preset-config

# bun
bun install -D @ruan-cat/vuepress-preset-config

# deno
deno install --dev npm:@ruan-cat/vuepress-preset-config
```

<!-- /automd -->
