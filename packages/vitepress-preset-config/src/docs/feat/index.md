# 功能

本 vitepress 预设配置提供以下基础功能：

## setUserConfig 的 extraConfig

当你需要调整预设内部的插件配置，或者覆盖默认的 Teek 主题配置时，请阅读 [setUserConfig 的 extraConfig 使用文档](./set-user-config-extra-config.md)。

## 图片放大预览

图片放大预览功能，来自于二次包装的 [vitepress-theme-teek](https://github.com/Kele-Bingtang/vitepress-theme-teek) 主题。
