# 增加切换 vitepress 主题的能力

我希望 `@ruan-cat/vitepress-preset-config` 增加一个新功能，通过点击导航栏提供的主题切换按钮，实现 vitepress 主题的直接切换。

默认主题为 `vitepress-theme-teek` ，可以通过按钮随时切换到 `@voidzero-dev/vitepress-theme` 主题。

开发该功能时，请预留好随时增加集成新的 vitepress 主题的空间。未来很有可能会集成多款 vitepress 主题。

集成 `@voidzero-dev/vitepress-theme` 主题的具体例子： https://github.com/vitejs/vite-plugin-registry/pull/11/files

这是一个富有挑战性的开发任务，请你充分发挥想象力完成开发。

1. 做好兼容性改造，确保本包原来的功能就正常可用。
