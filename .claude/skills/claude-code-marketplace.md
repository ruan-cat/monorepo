# 给 claude code 插件商城升级版本并编写更新日志

## 相关文件目录

- `.claude-plugin`
- `claude-code-marketplace\common-tools`

## 需要更新插件版本号的文件

在你更新 claude code 插件商城的版本号时，请在以下文件内更新版本号：

- `.claude-plugin\marketplace.json`
- `claude-code-marketplace\common-tools\.claude-plugin\plugin.json`

## 更新日志文件

每次更新的内容，都应该写在以下更新日志内。

- `claude-code-marketplace\common-tools\CHANGELOG.md`
