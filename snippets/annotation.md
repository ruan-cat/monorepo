## annotation
通用的注释模板。
```
/** $1 */
```

## annotation-toset
通用的注释模板。多了一个额外的占位符，表示应该替换并填充值。
```typescript,javascript
/** toset */
```

## annotation-toset
通用的注释模板。特指常用的 jsdoc @type 的模板。主要用在给单个变量导入 typescript 类型。
```javascript
/** @type { $1 } */
```
