# 包纳管与风险确认（模板）

在落盘前由用户或执行者确认。默认值必须显式写出。

## 版本模式

| 问题                                                     | 用户选择 | 备注                                                            |
| -------------------------------------------------------- | -------- | --------------------------------------------------------------- |
| 是否需要「各子包独立版本线」？                           | 是 / 否  | 是 → 优先 `independent`                                         |
| 若否，是否接受「统一版本线」或「仅 bump 变更包」的语义？ |          | 与 `selective` 语义对齐，见 references/version-mode-decision.md |

## 纳管范围

| --------------------------- | -------------- |
| 是否覆盖全部 workspace 包？ | 是 / 否 / 部分 |
| 若部分，glob 或列表是？     |                |
