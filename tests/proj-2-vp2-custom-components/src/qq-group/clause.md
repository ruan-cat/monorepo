# QQ 群条款

全体小爱丽丝必须遵守以下龟定：

## 不许涩涩 <Badge text="禁止" type="danger" />

涩涩 = 散播不良信息。不许涩涩。

## 不许催促 <Badge text="禁止" type="danger" />

催促 = 散播焦虑，不许催促钻头做以下事情：

- 画画
- 写插件
- 写 markdown 格式的文档

## 不许发感叹号 <Badge text="危险" type="warning" />

群主对感叹号"!"特别敏感，不要随便用感叹号，这与在群里大喊大叫没区别。

## 违规惩罚

禁言不听话的小爱丽丝半天。
