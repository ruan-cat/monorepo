## git仓库用户名

在克隆到本地的云效代码仓库内，更改自己的本地仓库git用户名。

1. 去信息表内补全`git仓库用户名`一栏

命名规则如下：

- 与各个平台的用户名保持一致。即满足[`明确自己统一的用户名`](../2025-5-11/index.md#明确自己统一的用户名)要求。
- 用户名可以保持这样的差异：必须严格增加f1-xxx格式的**前缀**。以 `EvLast` 举例说明，那么本地仓库的用户名就必须为 `f1-EvLast` 。
